# Cash Register OOP Project
  This project is a simple Cash Register built using Object Oriented Programming  in Python


# Features
 -Add items to the cash register
 -Calculate total price automatically
 -Apply percentage discounts
 -Void (cancel) the last transaction
 -Track transaction history

# How It Works
The project uses a CashRegister class with
  # Attributes
  discount -Percentage discount applied to total price
  total - Current total price in the register 
  items - List of items purchased
  previous_transactions - Transaction history

  # Methods
  Add Item - Adds items to the register and updates total price.
  Apply Discount - Applies percentage discount to total price.
  Void Last Transaction - Cancels the last transaction and updates totals.
